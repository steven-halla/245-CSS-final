Why react Native

I love programming with React native for a lot of reasons. First mobile uses a more modern approach, compare to HTML which is old and outdated. Mobile is also more secure than web. With React Native itself, the code applies to both IOS and android at the same time, you only have to worry on library conflicts at times so it's worth it.

Why React

One of the things that I love with modern functional component React is the hooks. Handling state is a breeze, whether it's global or local state. There isn't a bunch of boiler plate code, like with Redux, so your code will look cleaner. Other hooks include useEffect, which will perform functions as the webpage loads.
